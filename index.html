Welcome


Welcome to the home of the Monkey-Wrench project. Monkey-Wrench was designed with the sole purpose of being a lightweight, portable and fast C++ API.


License

/br> Monkey-Wrench is created with the MIT license to ensure that anyone and everyone can try it out freely.




Builds

/br> Monkey-Wrench works on most any Windows or UNIX based systems. Windows compiling tests have been performed on Windows 7 and Windows 8. UNIX compiling tests have been performed on Linux Mint 13. In all cases, Monkey-Wrench has been full compatible.
